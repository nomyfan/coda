# Demo workspace instructions

These are the root workspace's custom instructions. They are injected into every
agent rooted at this workspace as the **workspace-knowledge** segment (alongside
this workspace's skills), and hot-reload when this file changes — no restart.

## Conventions

- This is a demo project. Keep changes small and well-explained.
- Prefer the built-in tools (`read_file`, `grep`, `glob`) over shelling out.
- When you finish a task, summarize what changed in one short paragraph.
