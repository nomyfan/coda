# Changelog entry

Write changelog entries in this shape:

- Group under `Added`, `Changed`, `Fixed`, or `Removed`.
- One line per change, imperative mood ("Add per-agent workspace support").
- Link the PR or issue when there is one.

This skill is only visible to agents rooted at the `docs/` workspace — it shows
that per-agent workspaces carry their own skills, not just the root's.
